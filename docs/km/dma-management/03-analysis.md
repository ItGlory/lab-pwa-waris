# การวิเคราะห์ DMA (DMA Analysis)

## ภาพรวมการวิเคราะห์

การวิเคราะห์ DMA มีเป้าหมายเพื่อ:
- ระบุปริมาณน้ำสูญเสียที่แท้จริง
- แยกประเภทน้ำสูญเสีย (Physical vs Commercial)
- จัดลำดับความสำคัญในการแก้ไข
- ประเมินความคุ้มค่าการลงทุน

### การคำนวณ Water Balance

```
System Input Volume (SIV)
├── Authorized Consumption
│   ├── Billed Consumption
│   │   ├── Billed Metered (อ่านจากมิเตอร์)
│   │   └── Billed Unmetered (เหมาจ่าย)
│   └── Unbilled Consumption
│       ├── Unbilled Metered (มิเตอร์ราชการบางแห่ง)
│       └── Unbilled Unmetered (ดับเพลิง ล้างท่อ)
└── Water Losses = SIV - Authorized Consumption
    ├── Apparent Losses
    │   ├── Unauthorized Consumption (ลักน้ำ)
    │   └── Metering Inaccuracies (มิเตอร์ผิดพลาด)
    └── Real Losses = Water Losses - Apparent Losses
        ├── Leakage on Transmission (รั่วท่อเมน)
        ├── Leakage on Distribution (รั่วท่อจ่าย)
        └── Leakage on Service Connection (รั่วท่อบริการ)
```

## 2. Component Analysis (การวิเคราะห์องค์ประกอบ)

### 2.1 การประเมิน Real Losses

#### วิธี Top-Down (จาก MNF)

```
Real Losses = (MNF - Legitimate Night Use) × Hour Factor × Days

โดย:
- MNF = อัตราไหลต่ำสุดกลางคืน (ลบ.ม./ชม.)
- Legitimate Night Use = 0.6-1.5 ลิตร/ราย/ชม.
- Hour Factor = 24 ชม./วัน
- Days = จำนวนวัน
```

#### วิธี Bottom-Up (จากการสำรวจ)

```
Real Losses = (รั่วจากท่อเมน) + (รั่วจากท่อจ่าย) + (รั่วจากท่อบริการ)

รั่วแต่ละประเภท = จำนวนจุดรั่ว × อัตรารั่วเฉลี่ย × เวลารั่ว
```

### 2.2 การประเมิน Apparent Losses

#### ความผิดพลาดของมิเตอร์
```
Meter Inaccuracy Loss = ∑ (ปริมาณใช้จริง - ปริมาณมิเตอร์อ่านได้)

ค่าเฉลี่ย:
- มิเตอร์ใหม่ (<3 ปี): -2% ถึง +2%
- มิเตอร์เก่า (3-7 ปี): -5% ถึง 0%
- มิเตอร์เก่ามาก (>7 ปี): -15% ถึง -5%
```

#### การลักน้ำ
```
Theft Estimation = จำนวนรายสงสัย × ปริมาณเฉลี่ยต่อราย

ข้อบ่งชี้:
- การใช้น้ำต่ำผิดปกติ (<5 ลบ.ม./เดือน สำหรับครอบครัว)
- ไม่มีการใช้น้ำติดต่อกันหลายเดือน
- ประวัติการดัดแปลงมิเตอร์
```

## 3. Infrastructure Leakage Index (ILI)

### คำจำกัดความ

**ILI** คือดัชนีวัดประสิทธิภาพการจัดการน้ำสูญเสียทางกายภาพ

```
ILI = CARL / UARL

โดย:
- CARL = Current Annual Real Losses (น้ำสูญเสียจริงต่อปี)
- UARL = Unavoidable Annual Real Losses (น้ำสูญเสียที่หลีกเลี่ยงไม่ได้)
```

### การคำนวณ UARL (สูตร IWA)

```
UARL (ลิตร/วัน) = (18 × Lm + 0.80 × Nc + 25 × Lc) × P

โดย:
- Lm = ความยาวท่อเมน (กม.)
- Nc = จำนวนท่อบริการ (ราย)
- Lc = ความยาวท่อบริการรวม (กม.)
- P = แรงดันเฉลี่ย (เมตร)
```

### การตีความค่า ILI

| ILI | ระดับ | ความหมาย | การดำเนินการ |
|-----|------|---------|-------------|
| 1-2 | A | ดีเยี่ยม | รักษาระดับ |
| 2-4 | B | ดี | ปรับปรุงได้เล็กน้อย |
| 4-8 | C | ปานกลาง | ควรปรับปรุง |
| > 8 | D | ต้องปรับปรุง | ดำเนินการเร่งด่วน |

### ตัวอย่างการคำนวณ ILI

```
สมมติ DMA มี:
- ความยาวท่อเมน: 15 กม.
- จำนวนราย: 1,500 ราย
- ความยาวท่อบริการ: 12 กม.
- แรงดันเฉลี่ย: 25 เมตร

UARL = (18×15 + 0.80×1,500 + 25×12) × 25
     = (270 + 1,200 + 300) × 25
     = 1,770 × 25
     = 44,250 ลิตร/วัน
     = 16,161 ลบ.ม./ปี

ถ้า CARL = 48,000 ลบ.ม./ปี
ILI = 48,000 / 16,161 = 2.97 (ระดับ B)
```

## 4. Economic Analysis (การวิเคราะห์เศรษฐศาสตร์)

### 4.1 Economic Level of Leakage (ELL)

**ELL** คือระดับน้ำรั่วที่เหมาะสมทางเศรษฐศาสตร์
คือจุดที่ต้นทุนการลดรั่วเพิ่มขึ้นเท่ากับมูลค่าน้ำที่ประหยัดได้

```
         ต้นทุน
           │
           │      ╱ ต้นทุนน้ำรั่ว
           │     ╱
           │    ╱
           │   ╱      ╲ ต้นทุนลดรั่ว
           │  ╱         ╲
           │ ╱            ╲
           │╱              ╲
           └─────────────────────
                   ↑
                  ELL
              (จุดที่คุ้มค่า)
```

### 4.2 การคำนวณความคุ้มค่า

#### ต้นทุนน้ำสูญเสีย
```
ต้นทุนน้ำสูญเสียต่อปี = ปริมาณน้ำสูญเสีย × ต้นทุนน้ำต่อหน่วย

ตัวอย่าง:
- น้ำสูญเสีย: 100,000 ลบ.ม./ปี
- ต้นทุนการผลิต: 15 บาท/ลบ.ม.
- ต้นทุนน้ำสูญเสีย = 100,000 × 15 = 1,500,000 บาท/ปี
```

#### ROI ของการลงทุนลดน้ำสูญเสีย
```
ROI = (มูลค่าน้ำที่ประหยัดได้ / ต้นทุนการลงทุน) × 100

ตัวอย่าง: ติดตั้ง PRV
- ต้นทุนติดตั้ง: 200,000 บาท
- ลดน้ำสูญเสีย: 20,000 ลบ.ม./ปี
- มูลค่าน้ำประหยัด: 20,000 × 15 = 300,000 บาท/ปี
- ROI = (300,000 / 200,000) × 100 = 150%
- Payback Period = 200,000 / 300,000 = 0.67 ปี (8 เดือน)
```

## คำถามที่พบบ่อย (FAQ)

### Q: Water Balance คืออะไร?
**A:** Water Balance คือการวิเคราะห์สมดุลน้ำในระบบจ่ายน้ำ
แสดงให้เห็นว่าน้ำที่ผลิตแล้วไปอยู่ที่ไหน แบ่งเป็นน้ำจำหน่าย น้ำใช้ฟรี และน้ำสูญเสีย
เป็นพื้นฐานสำคัญในการวางแผนลดน้ำสูญเสีย

### Q: ILI คืออะไร และค่าเท่าไหร่ถือว่าดี?
**A:** ILI (Infrastructure Leakage Index) คือดัชนีวัดประสิทธิภาพการจัดการน้ำสูญเสีย
เป็นอัตราส่วนระหว่างน้ำสูญเสียจริง (CARL) กับน้ำสูญเสียที่หลีกเลี่ยงไม่ได้ (UARL)
- ILI 1-2: ดีเยี่ยม
- ILI 2-4: ดี
- ILI 4-8: ปานกลาง
- ILI > 8: ต้องปรับปรุง

### Q: UARL คำนวณอย่างไร?
**A:** UARL คำนวณจากสูตร IWA:
`UARL = (18×Lm + 0.80×Nc + 25×Lc) × P`
โดย Lm=ความยาวท่อเมน, Nc=จำนวนราย, Lc=ความยาวท่อบริการ, P=แรงดัน

### Q: ควรเริ่มวิเคราะห์ DMA จากตรงไหน?
**A:** ควรเริ่มจาก:
1. Water Balance Analysis: รู้ว่าน้ำหายไปไหน
2. Component Analysis: แยก Real Losses vs Apparent Losses
3. ILI Calculation: วัดประสิทธิภาพการจัดการ
4. Economic Analysis: จัดลำดับความสำคัญ

### Q: ELL (Economic Level of Leakage) คืออะไร?
**A:** ELL คือระดับน้ำรั่วที่เหมาะสมทางเศรษฐศาสตร์
เป็นจุดที่ต้นทุนการลดรั่วเพิ่มขึ้นเท่ากับมูลค่าน้ำที่ประหยัดได้
การลดรั่วต่ำกว่า ELL จะไม่คุ้มค่าทางเศรษฐศาสตร์
