# พื้นที่จ่ายน้ำย่อย (District Metered Area - DMA)

## คำจำกัดความ

**DMA (District Metered Area)** คือพื้นที่จ่ายน้ำที่แยกออกมาจากระบบใหญ่
โดยมีมิเตอร์วัดปริมาณน้ำเข้า-ออก เพื่อติดตามและควบคุมน้ำสูญเสียได้อย่างมีประสิทธิภาพ

## หลักการของ DMA

### ลักษณะของ DMA ที่ดี

1. **ขอบเขตชัดเจน**: มีวาล์วปิดกั้นรอบด้าน
2. **จุดเข้าน้ำน้อย**: 1-2 จุด ควบคุมง่าย
3. **มิเตอร์ครบ**: วัดน้ำเข้าทุกจุด
4. **ขนาดเหมาะสม**: 500-3,000 ราย/DMA

### ประโยชน์ของ DMA

- **ตรวจจับรั่วเร็ว**: เห็นการเปลี่ยนแปลงทันที
- **หาตำแหน่งรั่ว**: แคบพื้นที่ค้นหา
- **ควบคุมแรงดัน**: จัดการแยกเฉพาะพื้นที่
- **วัดผลได้**: เปรียบเทียบก่อน-หลัง

## องค์ประกอบของ DMA

```
┌─────────────────────────────────────┐
│              DMA                     │
│  ┌─────────────────────────────┐    │
│  │     พื้นที่บริการ            │    │
│  │   (500-3,000 ราย)           │    │
│  └─────────────────────────────┘    │
│                                      │
│  [M] ← มิเตอร์ DMA (วัดน้ำเข้า)      │
│  [V] ← วาล์วปิดกั้น (Boundary Valve) │
│  [P] ← เครื่องวัดแรงดัน (PRV/Logger) │
└─────────────────────────────────────┘
```

## ตัวชี้วัดสำคัญของ DMA

### 1. Minimum Night Flow (MNF)

อัตราการไหลต่ำสุดช่วงกลางคืน (02:00-04:00 น.)

- **MNF ต่ำ**: น้ำสูญเสียน้อย
- **MNF สูง**: มีการรั่วไหล

```
MNF = Q total - Q legitimate night use
น้ำสูญเสียโดยประมาณ ≈ MNF × 24 ชม. × 365 วัน
```

### 2. Net Night Flow (NNF)

```
NNF = MNF - Background Leakage - Legitimate Night Use
```

### 3. System Input Volume (SIV)

ปริมาณน้ำเข้า DMA ทั้งหมดต่อวัน/เดือน

### 4. NRW Rate

```
NRW (%) = ((SIV - Billed Consumption) / SIV) × 100
```

## ขนาดมาตรฐาน DMA

| ขนาด | จำนวนราย    | ความยาวท่อ | หมายเหตุ             |
| ---- | ----------- | ---------- | -------------------- |
| เล็ก | 500-1,000   | 5-10 กม.   | ควบคุมง่าย ต้นทุนสูง |
| กลาง | 1,000-2,500 | 10-25 กม.  | แนะนำ                |
| ใหญ่ | 2,500-5,000 | 25-50 กม.  | ยากควบคุม            |

## ขั้นตอนการจัดตั้ง DMA

### Phase 1: สำรวจและวางแผน

1. ศึกษาแผนที่ระบบท่อ
2. กำหนดขอบเขต DMA
3. หาจุดติดตั้งมิเตอร์

### Phase 2: ติดตั้งอุปกรณ์

1. ติดตั้งมิเตอร์ DMA
2. ติดตั้งวาล์วปิดกั้น
3. ติดตั้ง Data Logger

### Phase 3: ทดสอบและปรับแต่ง

1. ทดสอบการแยกน้ำ
2. ตรวจสอบความสมบูรณ์
3. ปรับแต่งขอบเขต

### Phase 4: ติดตามผล

1. เก็บข้อมูล MNF รายวัน
2. วิเคราะห์แนวโน้ม
3. ดำเนินการแก้ไข

## การติดตาม DMA

### ระบบ SCADA

- อ่านค่าอัตโนมัติ real-time
- แจ้งเตือนเมื่อผิดปกติ
- เก็บประวัติข้อมูล

### Data Logger

- บันทึกทุก 15 นาที
- ดาวน์โหลดข้อมูลสัปดาห์ละครั้ง
- วิเคราะห์ offline

### การแจ้งเตือน

- MNF เพิ่มขึ้น > 20%
- แรงดันผิดปกติ
- มิเตอร์ไม่ส่งข้อมูล

## เป้าหมาย DMA ที่ดี

| ตัวชี้วัด          | เป้าหมาย         |
| ------------------ | ---------------- |
| NRW Rate           | < 20%            |
| MNF per connection | < 5 ลิตร/ราย/ชม. |
| ILI                | < 4              |
| Response time      | < 24 ชม.         |

## คำถามที่พบบ่อย (FAQ)

### Q: DMA คืออะไร?

**A:** DMA (District Metered Area) หรือพื้นที่จ่ายน้ำย่อย คือพื้นที่จ่ายน้ำที่แยกออกมาจากระบบใหญ่ โดยมีมิเตอร์วัดปริมาณน้ำเข้า-ออก เพื่อติดตามและควบคุมน้ำสูญเสียได้อย่างมีประสิทธิภาพ

### Q: DMA ควรมีขนาดเท่าไหร่?

**A:** ขนาด DMA ที่แนะนำคือ 1,000-2,500 ราย (ขนาดกลาง) ความยาวท่อประมาณ 10-25 กม. ถ้าเล็กเกินไปต้นทุนสูง ถ้าใหญ่เกินไปยากต่อการควบคุม

### Q: MNF คืออะไร และทำไมถึงสำคัญ?

**A:** MNF (Minimum Night Flow) คืออัตราการไหลต่ำสุดในช่วงกลางคืน (02:00-04:00 น.) เป็นช่วงที่คนใช้น้ำน้อยที่สุด ดังนั้นน้ำที่ไหลส่วนใหญ่จึงเป็นน้ำรั่วไหล ทำให้ MNF เป็นตัวชี้วัดน้ำสูญเสียทางกายภาพที่ดีที่สุด

### Q: การจัดตั้ง DMA มีขั้นตอนอย่างไร?

**A:** การจัดตั้ง DMA มี 4 ขั้นตอนหลัก:
1. สำรวจและวางแผน - กำหนดขอบเขต
2. ติดตั้งอุปกรณ์ - มิเตอร์ วาล์ว Data Logger
3. ทดสอบ - ตรวจสอบความสมบูรณ์
4. ติดตามผล - เก็บข้อมูล MNF รายวัน

### Q: DMA ช่วยลดน้ำสูญเสียได้อย่างไร?

**A:** DMA ช่วยให้ตรวจจับการรั่วไหลได้เร็ว เพราะเห็นการเปลี่ยนแปลงของ MNF ทันที เมื่อมีท่อรั่วใหม่ MNF จะเพิ่มขึ้น ทำให้แจ้งเตือนและซ่อมได้เร็ว นอกจากนี้ยังช่วยแคบพื้นที่ค้นหาจุดรั่วได้ดีขึ้น
