# กระบวนการดำเนินงาน กปภ. (PWA Operational Processes)

## ภาพรวมกระบวนการ

```
แหล่งน้ำดิบ → สูบน้ำดิบ → ผลิตน้ำประปา → ส่งจ่ายน้ำ → ผู้ใช้น้ำ
```

---

## 1. กระบวนการผลิตน้ำประปา (Water Treatment Process)

### 1.1 การสูบน้ำดิบ (Raw Water Intake)

#### แหล่งน้ำดิบ
| ประเภท | ตัวอย่าง |
|--------|--------|
| แม่น้ำ | เจ้าพระยา, ชี, มูล, ปิง |
| อ่างเก็บน้ำ | เขื่อนภูมิพล, ลำตะคอง |
| บ่อบาดาล | ในพื้นที่ไม่มีแหล่งผิวดิน |
| คลอง | คลองชลประทาน |

#### คุณภาพน้ำดิบที่ต้องติดตาม
- ความขุ่น (Turbidity)
- สี (Color)
- pH
- ความเป็นด่าง (Alkalinity)
- สารแขวนลอย (Suspended Solids)

### 1.2 การปรับปรุงคุณภาพน้ำ (Water Treatment)

#### ขั้นตอนการผลิต

```
น้ำดิบ
   │
   ▼
┌─────────────────┐
│ 1. Coagulation  │  ← สารส้ม (Alum) / PAC
│    การทำให้รวมตัว │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 2. Flocculation │  ← กวนช้าๆ ให้ตกตะกอน
│    การตกตะกอน   │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 3. Sedimentation│  ← ถังตกตะกอน
│    การทำให้ใส   │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 4. Filtration   │  ← กรองผ่านทราย/ถ่าน
│    การกรอง      │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 5. Disinfection │  ← คลอรีน / UV
│    การฆ่าเชื้อ   │
└────────┬────────┘
         │
         ▼
   น้ำประปา
```

#### สารเคมีที่ใช้ในการผลิต
| สารเคมี | วัตถุประสงค์ |
|---------|-------------|
| สารส้ม (Alum) | ทำให้ตกตะกอน |
| PAC (Poly Aluminium Chloride) | ทำให้ตกตะกอน |
| ปูนขาว (Lime) | ปรับ pH |
| คลอรีน (Chlorine) | ฆ่าเชื้อโรค |
| โซดาแอช (Soda Ash) | ปรับ pH |

### 1.3 มาตรฐานคุณภาพน้ำประปา

#### ตามมาตรฐาน กรมอนามัย
| พารามิเตอร์ | มาตรฐาน |
|------------|---------|
| ความขุ่น | ไม่เกิน 4 NTU |
| สี | ไม่เกิน 15 หน่วยแพลทินัม-โคบอลต์ |
| pH | 6.5 - 8.5 |
| คลอรีนคงเหลือ | 0.2 - 0.5 mg/L |
| เหล็ก | ไม่เกิน 0.5 mg/L |
| แมงกานีส | ไม่เกิน 0.3 mg/L |
| โคลิฟอร์มแบคทีเรีย | ไม่พบ/100 mL |

## 2. กระบวนการส่งจ่ายน้ำ (Water Distribution Process)

### 2.1 ระบบส่งน้ำ (Transmission)

#### องค์ประกอบหลัก
```
โรงกรอง → ท่อส่งน้ำ → ถังเก็บน้ำ/หอถังสูง → ท่อจ่ายน้ำ → ผู้ใช้น้ำ
```

| องค์ประกอบ | หน้าที่ |
|-----------|--------|
| สถานีสูบน้ำดี | เพิ่มแรงดันส่งน้ำ |
| ถังเก็บน้ำใส | พักน้ำก่อนจ่าย |
| หอถังสูง | เก็บน้ำ + รักษาแรงดัน |
| ท่อส่งน้ำเมน | ส่งน้ำระยะไกล |

### 2.2 ระบบจ่ายน้ำ (Distribution)

#### ประเภทท่อ
| ประเภท | ขนาด | การใช้งาน |
|--------|------|----------|
| ท่อเมน (Main) | 300-1,500 มม. | ส่งน้ำหลัก |
| ท่อจ่าย (Distribution) | 100-300 มม. | จ่ายในพื้นที่ |
| ท่อบริการ (Service) | 15-50 มม. | เข้าบ้าน |

#### วัสดุท่อ
| วัสดุ | ข้อดี | ข้อเสีย |
|-------|------|--------|
| HDPE | ยืดหยุ่น ไม่รั่ว | ราคาสูง |
| PVC | ราคาถูก ติดตั้งง่าย | เปราะ |
| Ductile Iron | แข็งแรง ทนแรงดัน | หนัก กัดกร่อน |
| เหล็ก (Steel) | แข็งแรง | สนิม ต้องเคลือบ |
| AC (Asbestos Cement) | ราคาถูก | เปราะ ล้าสมัย |

### 2.3 การควบคุมแรงดัน (Pressure Management)

#### ระดับแรงดันมาตรฐาน
| ช่วงเวลา | แรงดัน (bar) |
|---------|-------------|
| กลางวัน (พีค) | 1.5 - 3.0 |
| กลางคืน | 1.0 - 2.0 |
| ต่ำสุดที่รับได้ | > 0.7 |

#### อุปกรณ์ควบคุมแรงดัน
- **PRV** (Pressure Reducing Valve): ลดแรงดัน
- **PSV** (Pressure Sustaining Valve): รักษาแรงดันขั้นต่ำ
- **BPT** (Break Pressure Tank): ถังลดแรงดัน

## 3. กระบวนการจัดการน้ำสูญเสีย (NRW Management Process)

### 3.1 การจัดตั้ง DMA

```
1. สำรวจพื้นที่
   │
   ▼
2. ออกแบบ DMA
   │
   ▼
3. ติดตั้งมิเตอร์ + วาล์ว
   │
   ▼
4. ทดสอบความสมบูรณ์
   │
   ▼
5. เก็บข้อมูล MNF
   │
   ▼
6. วิเคราะห์น้ำสูญเสีย
```

### 3.2 การสำรวจรั่วเชิงรุก (Active Leak Detection)

#### ขั้นตอน
1. **Night Flow Analysis**: วิเคราะห์ MNF ผิดปกติ
2. **Step Testing**: ปิด-เปิดวาล์วหาพื้นที่รั่ว
3. **Acoustic Survey**: ฟังเสียงรั่ว
4. **Pinpointing**: หาตำแหน่งแม่นยำ
5. **Repair**: ซ่อมแซม
6. **Verify**: ตรวจสอบหลังซ่อม

### 3.3 การซ่อมบำรุง (Maintenance)

#### ประเภทการซ่อมบำรุง
| ประเภท | คำอธิบาย | ตัวอย่าง |
|--------|---------|---------|
| ซ่อมฉุกเฉิน (Emergency) | แก้ไขเร่งด่วน | ท่อแตก |
| ซ่อมบำรุงเชิงป้องกัน (Preventive) | ตามกำหนดเวลา | ล้างถัง, ตรวจวาล์ว |
| ซ่อมบำรุงเชิงคาดการณ์ (Predictive) | ตามสภาพ | เปลี่ยนท่อก่อนแตก |

## 6. ระบบ DMAMA (DMA Management Application)

### 6.1 ภาพรวม

**DMAMA** เป็นระบบบริหารจัดการ DMA ของ กปภ. ใช้ติดตามน้ำสูญเสียทั่วประเทศ

### 6.2 ข้อมูลที่บันทึก

| ข้อมูล | ความถี่ | หน่วย |
|--------|--------|------|
| System Input | รายวัน | ลบ.ม./วัน |
| Billed Consumption | รายเดือน | ลบ.ม./เดือน |
| NRW Rate | รายเดือน | % |
| MNF | รายวัน | ลิตร/ชม. |
| แรงดัน | รายวัน | bar |
| จำนวนผู้ใช้น้ำ | รายเดือน | ราย |

### 6.3 รายงานที่สร้าง

- รายงาน NRW รายเดือน
- รายงาน MNF รายสัปดาห์
- รายงานแนวโน้มน้ำสูญเสีย
- รายงานการซ่อมรั่ว
- รายงานการเปลี่ยนท่อ

## คำถามที่พบบ่อย (FAQ)

### Q: น้ำประปาผลิตจากอะไร?
**A:** น้ำประปา กปภ. ผลิตจากน้ำดิบ (แม่น้ำ อ่างเก็บน้ำ หรือบ่อบาดาล)
ผ่านกระบวนการปรับปรุงคุณภาพ ได้แก่ ตกตะกอน กรอง และฆ่าเชื้อด้วยคลอรีน

### Q: ทำไมน้ำประปาถึงมีกลิ่นคลอรีน?
**A:** คลอรีนใส่เพื่อฆ่าเชื้อโรคและรักษาคุณภาพน้ำระหว่างส่งผ่านท่อ
ปริมาณคลอรีนที่ใส่อยู่ในระดับปลอดภัย (0.2-0.5 mg/L) ตามมาตรฐานกรมอนามัย

### Q: ระบบ DMAMA คืออะไร?
**A:** DMAMA (DMA Management Application) เป็นระบบบริหารจัดการ DMA ของ กปภ.
ใช้บันทึกและติดตามข้อมูลน้ำสูญเสียของแต่ละ DMA ทั่วประเทศ
ประกอบด้วยข้อมูล System Input, NRW Rate, MNF และแรงดัน

### Q: การอ่านมิเตอร์ทำบ่อยแค่ไหน?
**A:** โดยทั่วไป กปภ. อ่านมิเตอร์เดือนละ 1 ครั้ง ยกเว้นผู้ใช้น้ำรายใหญ่
อาจอ่านบ่อยกว่า ปัจจุบันมีการใช้ Smart Meter ที่ส่งข้อมูลอัตโนมัติในบางพื้นที่

### Q: ค่าน้ำคิดอย่างไร?
**A:** ค่าน้ำคิดตามปริมาณการใช้ (ลบ.ม.) x อัตราค่าน้ำ (บาท/ลบ.ม.)
โดยอัตราค่าน้ำแตกต่างตามประเภทผู้ใช้น้ำ (ที่อยู่อาศัย ราชการ ธุรกิจ อุตสาหกรรม)
และมีค่าบริการรายเดือนเพิ่มเติม
