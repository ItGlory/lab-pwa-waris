# น้ำสูญเสียทางกายภาพ (Physical Losses)

## คำจำกัดความ

**น้ำสูญเสียทางกายภาพ** หรือ **Real Losses** คือน้ำที่รั่วไหลออกจากระบบท่อจ่ายน้ำ
ก่อนถึงมิเตอร์ผู้ใช้น้ำ

## ประเภทของการรั่วไหล

### 1. รั่วไหลจากท่อเมน (Mains Leakage)

- **ท่อแตก**: เกิดจากความเก่า แรงดันสูง หรือแรงกระแทก
- **รอยต่อรั่ว**: ข้อต่อหลวม ยางซีลเสื่อม
- **การกัดกร่อน**: ท่อเหล็กเป็นสนิม ท่อ PVC เปราะ

### 2. รั่วไหลจากท่อบริการ (Service Connection Leakage)

- **ท่อประปาเข้าบ้าน**: ท่อขนาดเล็ก รั่วยากตรวจพบ
- **ก่อนมิเตอร์**: รั่วแต่ไม่ผ่านมิเตอร์ ไม่มีค่าน้ำ

### 3. รั่วไหลจากถังเก็บและอุปกรณ์

- **ถังสูงล้น**: ลูกลอยเสีย ควบคุมระดับน้ำไม่ได้
- **วาล์วรั่ว**: ประตูน้ำปิดไม่สนิท
- **หัวดับเพลิง**: ซีลเสื่อม น้ำซึมตลอดเวลา

## ปัจจัยที่ทำให้เกิดการรั่วไหล

| ปัจจัย    | ผลกระทบ                   |
| --------- | ------------------------- |
| อายุท่อ   | ท่อเก่า > 30 ปี เสี่ยงสูง |
| วัสดุท่อ  | ท่อเหล็ก กัดกร่อนง่าย     |
| แรงดันน้ำ | แรงดันสูง เร่งการรั่ว     |
| สภาพดิน   | ดินทรุด กดทับท่อ          |
| การจราจร  | น้ำหนักรถกดทับ            |
| อุณหภูมิ  | ร้อน-เย็น ท่อขยาย-หดตัว   |

## การตรวจจับการรั่วไหล

### วิธี Passive

- **รอแจ้งเหตุ**: รอให้ประชาชนแจ้ง
- **ตรวจสอบมิเตอร์**: เปรียบเทียบยอดใช้น้ำ

### วิธี Active (แนะนำ)

- **Acoustic Listening**: ฟังเสียงน้ำรั่วด้วยเครื่องฟัง
- **Ground Microphone**: ตรวจจับเสียงใต้ดิน
- **Correlator**: หาตำแหน่งรั่วแม่นยำ
- **Step Test**: ปิด-เปิดวาล์วทีละส่วน
- **MNF Analysis**: วิเคราะห์อัตราไหลกลางคืน

## ดัชนีวัดน้ำสูญเสียทางกายภาพ

### ILI (Infrastructure Leakage Index)

```
ILI = CARL / UARL
```

- **CARL**: Current Annual Real Losses (น้ำสูญเสียจริงต่อปี)
- **UARL**: Unavoidable Annual Real Losses (น้ำสูญเสียที่หลีกเลี่ยงไม่ได้)

| ILI | ระดับ | คำอธิบาย               |
| --- | ----- | ---------------------- |
| 1-2 | A     | ดีเยี่ยม ยากจะลดได้อีก |
| 2-4 | B     | ดี มีโอกาสลดได้บ้าง    |
| 4-8 | C     | ปานกลาง ควรปรับปรุง    |
| > 8 | D     | ต้องดำเนินการด่วน      |

## แนวทางลดน้ำสูญเสียทางกายภาพ

1. **Active Leak Control**: สำรวจรั่วเชิงรุก
2. **Pressure Management**: ลดแรงดันน้ำ
3. **Pipeline Replacement**: เปลี่ยนท่อเก่า
4. **Speed of Repairs**: ซ่อมรั่วให้เร็ว
5. **DMA Management**: แบ่งพื้นที่ควบคุม
