# คำศัพท์ภาษาพูดและคำพ้องความหมาย

> เพื่อให้ RAG เข้าใจคำถามจากผู้ใช้ทั่วไปที่อาจไม่ใช้คำศัพท์เทคนิค

---

## น้ำสูญเสีย (NRW) - คำพ้องความหมาย

| คำทางการ | คำที่ใช้ทั่วไป | คำภาษาอังกฤษ |
|----------|---------------|--------------|
| น้ำสูญเสีย | น้ำหาย, น้ำไม่มีรายได้, น้ำที่ขาดทุน | NRW, water loss, lost water |
| น้ำสูญเสียทางกายภาพ | น้ำรั่ว, ท่อรั่ว, น้ำไหลทิ้ง | real loss, physical loss, leakage |
| น้ำสูญเสียเชิงพาณิชย์ | น้ำที่วัดไม่ได้, มิเตอร์ผิด, ขโมยน้ำ | apparent loss, commercial loss |
| ปริมาณน้ำเข้าระบบ | น้ำที่ผลิต, น้ำเข้า, น้ำจ่าย | system input, water production |

## คำถามที่ผู้ใช้อาจถาม (User Question Patterns)

### เกี่ยวกับน้ำสูญเสีย

```
คำถามทั่วไป:
- "น้ำหายไปไหน?" → ถามเรื่อง NRW
- "ทำไมน้ำสูญเสียเยอะ?" → ถามสาเหตุ NRW
- "น้ำรั่วเกิดจากอะไร?" → ถามสาเหตุ Real Loss
- "เปอร์เซ็นต์น้ำหายควรเท่าไหร่?" → ถามมาตรฐาน NRW
- "น้ำหายกี่เปอร์เซ็นต์ถือว่าเยอะ?" → ถามเกณฑ์ NRW
```

### เกี่ยวกับ DMA

```
คำถามทั่วไป:
- "DMA คืออะไร?" → อธิบาย DMA
- "ทำไมต้องมี DMA?" → ถามประโยชน์
- "โซนน้ำคืออะไร?" → ถามเรื่อง DMA
- "ทำ DMA ยังไง?" → ถามขั้นตอนจัดตั้ง
- "DMA ใหญ่แค่ไหนดี?" → ถามขนาดมาตรฐาน
```

### เกี่ยวกับมิเตอร์

```
คำถามทั่วไป:
- "มิเตอร์เดินช้าคืออะไร?" → Under-registration
- "ควรเปลี่ยนมิเตอร์เมื่อไหร่?" → อายุการใช้งาน
- "Smart Meter ดียังไง?" → ประโยชน์ Smart Meter
- "มิเตอร์ผิดเกิดจากอะไร?" → สาเหตุ Meter Error
- "ทดสอบมิเตอร์ยังไง?" → วิธีทดสอบ
```

### เกี่ยวกับการซ่อมแซม

```
คำถามทั่วไป:
- "หาท่อรั่วยังไง?" → Leak Detection
- "เครื่องหารั่วมีอะไรบ้าง?" → อุปกรณ์ตรวจจับ
- "ซ่อมท่อรั่วนานไหม?" → ระยะเวลาซ่อม
- "แจ้งท่อแตกที่ไหน?" → ช่องทางแจ้งเหตุ
- "ลดน้ำรั่วยังไง?" → แนวทางลด NRW
```

### เกี่ยวกับ กปภ.

```
คำถามทั่วไป:
- "กปภ.คืออะไร?" → ข้อมูลองค์กร
- "กปภ.มีกี่เขต?" → 10 เขต
- "โทรหาประปาเบอร์อะไร?" → 1662
- "ค่าน้ำแพงจัง" → อธิบายค่าน้ำ
- "น้ำประปาไม่มา" → แจ้งเหตุ
```

## ตัวอย่างการตอบคำถามภาษาพูด

### Q: "น้ำหายไปไหนหมด?"

**A:** น้ำที่ "หาย" หรือน้ำสูญเสีย (NRW) มี 3 สาเหตุหลัก:

1. **ท่อรั่ว** (60-70%) - น้ำรั่วจากท่อเก่า รอยต่อหลวม
2. **มิเตอร์ผิด** (20-25%) - มิเตอร์เก่าวัดน้ำได้น้อยกว่าจริง
3. **ลักน้ำ** (5-15%) - ต่อท่อเถื่อน ดัดแปลงมิเตอร์

### Q: "ทำไมค่าน้ำบ้านข้างๆ ถูกกว่า?"

**A:** อาจมีหลายสาเหตุ:

1. ใช้น้ำน้อยกว่าจริง
2. มิเตอร์บ้านเขาเดินช้า (ควรแจ้งเปลี่ยน)
3. มีท่อรั่วในบ้านคุณ (ตรวจสอบโดยปิดน้ำทุกจุดแล้วดูมิเตอร์)
4. จำนวนคนในบ้านต่างกัน

### Q: "รู้ได้ไงว่าท่อในบ้านรั่ว?"

**A:** วิธีตรวจสอบง่ายๆ:

1. ปิดก๊อกน้ำทุกจุดในบ้าน
2. ดูมิเตอร์น้ำ ถ้ายังหมุนอยู่ = มีท่อรั่ว
3. ตรวจหาจุดชื้นตามพื้น ผนัง
4. ฟังเสียงน้ำไหลเมื่อปิดน้ำหมด
5. ถ้าพบรั่วให้แจ้งช่าง หรือโทร 1662

## คำถามที่พบบ่อย (FAQ)

### Q: ทำไม RAG ต้องเข้าใจภาษาพูด?

**A:** ผู้ใช้ทั่วไปมักไม่ใช้คำศัพท์เทคนิค เช่น ถาม "น้ำหายไปไหน" แทน "NRW เกิดจากอะไร" การมีคำพ้องความหมายช่วยให้ RAG จับคู่คำถามกับเอกสารที่ถูกต้องได้

### Q: คำไหนที่ผู้ใช้ใช้บ่อยที่สุด?

**A:** จากการวิเคราะห์ คำที่ใช้บ่อย ได้แก่:

1. "น้ำรั่ว" - แทน Physical Loss
2. "มิเตอร์" - อุปกรณ์ที่ทุกคนรู้จัก
3. "ค่าน้ำ" - เรื่องที่กระทบผู้ใช้โดยตรง
4. "ประปา" - เรียกแทน กปภ.

### Q: ควรเพิ่มคำพ้องอะไรอีก?

**A:** สามารถเพิ่มได้ตามที่พบจากการใช้งานจริง โดยเฉพาะ:

1. คำท้องถิ่น/ภาษาถิ่น
2. คำที่พิมพ์ผิดบ่อย
3. คำย่อที่นิยมใช้
4. คำที่เกิดจาก Autocorrect ผิด
