# คำศัพท์เทคนิค (Technical Terms)

## คำศัพท์ด้านวิศวกรรมน้ำ

| ----------------- | ---------------- | ------------------------------------ |
| Transmission Main | ท่อส่งน้ำหลัก    | ท่อขนาดใหญ่ส่งน้ำจากโรงกรองไปถังเก็บ |
| Distribution Main | ท่อจ่ายหลัก      | ท่อขนาดกลางจ่ายน้ำในพื้นที่          |
| Service Line      | ท่อบริการ        | ท่อขนาดเล็กเข้าบ้านผู้ใช้            |
| Dead End          | ท่อตัน           | ปลายท่อที่ไม่มีทางออก                |
| Loop              | วงจรท่อ          | ท่อที่เชื่อมต่อเป็นวงกลม             |
| Cross Connection  | การเชื่อมต่อข้าม | การต่อท่อประปากับแหล่งน้ำอื่น (ห้าม) |

## คำถามที่พบบ่อย (FAQ)

### Q: ท่อ HDPE กับ PVC ต่างกันอย่างไร?

**A:** HDPE ยืดหยุ่นกว่า ข้อต่อหลอมรวม (ไม่รั่ว) แต่ราคาแพงกว่า
PVC ราคาถูก ติดตั้งง่าย แต่เปราะและข้อต่อใช้กาว/ยางซึ่งอาจรั่วได้

### Q: 1 bar เท่ากับกี่เมตรน้ำ?

**A:** 1 bar ≈ 10.2 เมตรน้ำ (m H₂O)
หรือพูดง่ายๆ ว่า 1 bar คือแรงดันที่สามารถดันน้ำขึ้นไปได้สูงประมาณ 10 เมตร

### Q: Q1, Q2, Q3, Q4 ของมิเตอร์หมายความว่าอย่างไร?

**A:** เป็นค่าอัตราไหลมาตรฐานของมิเตอร์ตาม ISO 4064:

- Q1: อัตราไหลต่ำสุดที่วัดได้แม่นยำ (±5%)
- Q2: อัตราไหลเปลี่ยนผ่าน (±2%)
- Q3: อัตราไหลปกติ (±2%)
- Q4: อัตราไหลสูงสุดที่รับได้ (±2%)

### Q: Correlator ทำงานอย่างไร?

**A:** Correlator ใช้เซ็นเซอร์ 2 ตัววางคนละด้านของจุดสงสัยรั่ว
บันทึกเสียงจากทั้งสองจุด แล้วคำนวณตำแหน่งรั่วจากความต่างของเวลาที่เสียงมาถึง
สามารถระบุตำแหน่งได้แม่นยำภายใน ±0.5 เมตร

### Q: SCADA กับ Data Logger ต่างกันอย่างไร?

**A:** Data Logger เป็นอุปกรณ์บันทึกข้อมูลอย่างเดียว อาจส่งข้อมูลหรือไม่ก็ได้
SCADA เป็นระบบควบคุมและเก็บข้อมูลครบวงจร สามารถควบคุมอุปกรณ์ระยะไกล
แสดงผล Real-time และแจ้งเตือนอัตโนมัติได้
