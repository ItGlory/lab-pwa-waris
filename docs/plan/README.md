## Design Language

The WARIS platform follows a modern, innovative design language:

- **Glassmorphism**: Frosted glass effects with backdrop blur
- **PWA Brand Colors**: Cyan (#00C2F3), Blue Deep (#065BAA), Navy (#0D2E5C)
- **Micro-interactions**: Scale, rotate, glow on hover
- **Animations**: Fade-in, blur-in, shimmer, float, breathing-glow
